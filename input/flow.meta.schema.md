# flow.meta.json Schema v1 規格說明

版本：flow-meta-v1\
適用系統：PydotStudio v2（Web 模式 + Static 模式相容）\
目的：記錄流程圖中「節點的實作層資訊」，不影響拓樸結構。

------------------------------------------------------------------------

# 一、設計目標

flow.meta.json 負責：

-   記錄每個節點的實作進度
-   記錄測試、文件、PR、證據等連結
-   支援專案完成度自動計算
-   可由 Web Viewer 編輯並匯出
-   不修改 flow.json 的拓樸

核心原則：

1.  與 flow.json 分離（Meta Separation）
2.  以 node_id 作為唯一關聯鍵
3.  可向下相容擴充（schema_version 控制）
4.  可支援加權完成度計算

------------------------------------------------------------------------

# 二、整體結構

``` json
{
  "schema_version": "flow-meta-v1",
  "project": { ... },
  "defaults": { ... },
  "nodes": { ... },
  "metrics": { ... }
}
```

------------------------------------------------------------------------

# 三、根層欄位說明

## 1. schema_version (必要)

``` json
"schema_version": "flow-meta-v1"
```

用途：未來版本升級時做相容性判斷。

------------------------------------------------------------------------

## 2. project（可選）

``` json
"project": {
  "name": "專案名稱",
  "description": "簡述",
  "owner": "負責團隊或人員",
  "updated_at": "2026-03-03T10:00:00+08:00",
  "tags": ["RAG", "Import"]
}
```

------------------------------------------------------------------------

## 3. defaults（可選）

``` json
"defaults": {
  "status": "todo",
  "progress": 0,
  "weight": 1
}
```

------------------------------------------------------------------------

## 4. nodes（必要）

``` json
"nodes": {
  "NODE_ID": { NodeMeta },
  "NODE_ID2": { NodeMeta }
}
```

Key 必須對應 flow.json 中的 node.id。

------------------------------------------------------------------------

# 四、NodeMeta 欄位說明

## status（列舉）

-   todo
-   doing
-   done
-   blocked
-   paused
-   deprecated
-   na

------------------------------------------------------------------------

## progress

-   型別：integer
-   範圍：0--100

------------------------------------------------------------------------

## weight

-   型別：number
-   範圍：\>= 0

------------------------------------------------------------------------

## links 範例

``` json
"links": [
  {
    "title": "CI 測試履歷",
    "url": "https://ci/job/123",
    "type": "test_report"
  }
]
```

------------------------------------------------------------------------

## tests 範例

``` json
"tests": [
  {
    "name": "unit",
    "script_ref": "repo:tests/test_x.py",
    "report_url": "https://ci/tests/123",
    "last_run_at": "2026-03-02T21:10:00+08:00",
    "pass_rate": 0.98
  }
]
```

------------------------------------------------------------------------

# 五、專案完成度計算規則

每個節點完成度：

node_completion = progress / 100

專案完成度（加權平均）：

project_completion = sum(weight × node_completion) / sum(weight)
